Bienvenue sur mon portfolio !

Je m’appelle Alex Balmes, développeur web en recherche d’alternance pour devenir full-stack.

Ce projet est une single-page application réalisée en HTML, CSS et JavaScript vanilla.
AlexXP simule un mini windows XP : gestion du démarrage (avec cookie pour fluidifier la navigation), comportements aléatoires de l’OS, et surtout une logique complète de fenêtres (ouvrir, minimiser, plein écran, drag and drop) grâce à une architecture modulaire et évolutive.

Ajouter une application est simple : une icône avec data-appname, un dossier dédié, et tout fonctionne instantanément.

Applications intégrées :

  • Convertisseur : créé en 2024 pour des calculs aéronautiques, puis largement retravaillé.
  • Simon : jeu JavaScript de 2023, adapté à la logique SPA.
  • StarSupply : Un deuxième jeu créé après le portfolio, avec sa propre API, et intégré à la page via un iframe depuis l’URL du projet : starsupply.alexbalmes.dev/, ce qui me permet d’utiliser également cette technologie.
  • Internet Explorer : mini-navigateur avec historique et affichage dynamique, développé en même temps que le portfolio.
  • Mon CV et Cette note.

Le « navigateur » permet aussi d’explorer trois projets :

  • Dex’s Gallery : tentative d’auto-hébergement serveur/API (reste aujourd’hui en statique).
  • Booki (2022) : mon premier projet, bases du HTML/CSS, responsive et intégration de maquette.
  • Oh My Food (2022) : deuxième projet d’intégration, initialement en SCSS (ici version CSS compilée).
